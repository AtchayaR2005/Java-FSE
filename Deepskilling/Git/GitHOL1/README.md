# Git HOL1 - Git Configuration and Repository Setup

## Objective

Learn the basic Git commands and configure Git on the local machine.

## Commands Executed

```bash
git --version
git config --global user.name "AtchayaR2005"
git config --global user.email "rj.atchi@gmail.com"
git config --global --list
git init
git status
echo "Welcome to Git" > welcome.txt
git add welcome.txt
git commit -m "Added welcome file"
git pull origin main
git push origin main
```

## Output

- Git configured successfully.
- Repository initialized.
- welcome.txt created.
- File committed successfully.
- Changes pushed to GitHub.
