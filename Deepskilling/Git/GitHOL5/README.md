# Git HOL5 - Clean Up and Push to Remote Repository

## Objective

- Clean the working directory.
- Pull the latest changes.
- Push local changes to the remote repository.

## Steps Performed

### 1. Verified repository status

```bash
git status
```

### 2. Listed all branches

```bash
git branch -a
```

### 3. Pulled the latest changes

```bash
git pull origin main
```

### 4. Added files

```bash
git add .
```

### 5. Committed the changes

```bash
git commit -m "Completed Git HOL5 - Clean Up and Push to Remote Repository"
```

### 6. Pushed the changes

```bash
git push origin main
```

### 7. Verified changes on GitHub

Opened the GitHub repository and confirmed that the latest commit was available.

## Result

Successfully synchronized the local repository with the remote GitHub repository.
